# Introducción a shadcn/ui

**shadcn/ui** NO es una librería de componentes. No se instala como una dependencia (`npm install @mui/material`).

Es una colección de componentes reutilizables que puedes copiar y pegar en tus aplicaciones.

## ¿Qué significa esto?

*   Tú eres dueño del código del componente.
*   Puedes personalizarlo como quieras editando el archivo `tsx` directamente.
*   No estás atado a las decisiones de diseño de una librería externa.
*   Usa **Tailwind CSS** para los estilos.
*   Usa **Radix UI** para la accesibilidad y lógica compleja (headless UI).

## ¿Por qué elegirlo?

1.  **Diseño Increíble:** Estéticamente pulido desde el primer día.
2.  **Accesible:** Construido sobre primitivas de Radix UI que garantizan soporte para lectores de pantalla y navegación por teclado.
3.  **Personalizable:** Como el código vive en tu proyecto, puedes cambiar cualquier cosa.

## Stack Tecnológico

Para usar shadcn/ui, necesitas:
*   React
*   Tailwind CSS
*   TypeScript (Recomendado)
*   Lucide React (para iconos)
